# 📱 Social Media Addiction Analysis Dashboard

An interactive **data analytics dashboard** built using **Streamlit, Plotly, and Machine Learning** to analyze social media usage patterns and their impact on students’ mental health and academic performance.

---

## 🚀 Features

* 📊 **Interactive Dashboard** with modern UI
* 🔍 **Data Explorer** with filtering options

* 📱 **Platform Analysis**
  * Usage by platform
  * Gender-based insights
* 🎓 **Academic Impact Analysis**
  * Usage vs academic performance
* 💡 **Insights & Recommendations**

---

## 🛠️ Tech Stack

* **Frontend/UI:** Streamlit
* **Visualization:** Plotly, Matplotlib, Seaborn
* **Backend/Data Processing:** Pandas, NumPy
* **Machine Learning:** Scikit-learn

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/YOUR_USERNAME/SMA-Dashboard.git
cd SMA-Dashboard
```

### 2️⃣ Create virtual environment (optional but recommended)

```
python -m venv venv
venv\Scripts\activate   # Windows
```

### 3️⃣ Install dependencies

```
pip install -r requirements.txt
```

### 4️⃣ Run the app

```
streamlit run app.py
```
